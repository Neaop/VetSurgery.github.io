🧿 第五章 眼科疾病选段

👁️ 5.1 眼睑内翻（Entropion）

📌 一、定义

眼睑内翻是指眼睑缘向内卷，睫毛接触角膜，对眼球造成刺激或损伤，常见于犬，尤其是面部皮肤松弛的犬种。

⚙️ 二、病因

  • 先天性（如沙皮犬、松狮犬）
  • 后天性（伤口瘢痕牵拉、老龄皮肤松弛）
  • 痉挛性（眼睑痉挛引起暂时性内翻）

👀 三、临床表现

  • 睫毛倒生刺激角膜，引起流泪、结膜充血、角膜炎症甚至溃疡
  • 动物频繁眨眼、抓挠眼睛、畏光

🛠️ 四、治疗

  • 轻度可采用临时缝合法或胶带外拉
  • 严重者应行手术矫正，如Hotz-Celsus法切除多余皮肤

👁️ 5.2 眼睑外翻（Ectropion）

📌 一、定义

眼睑边缘向外翻转，睑板暴露，无法贴合眼球，导致结膜外露和泪液外流。

⚙️ 二、病因

  • 先天性（如大丹犬、猎犬）
  • 老年犬睑肌松弛
  • 继发于慢性炎症或外伤手术后瘢痕牵拉

👀 三、临床表现

  • 睑缘外翻，结膜持续暴露
  • 泪液不能正常分布，引起结膜干燥、刺激性炎症

🛠️ 四、治疗

  • 温和外翻者使用润滑眼膏或滴眼液缓解不适
  • 明显者手术矫正，如V-Y推进术、睑缘收缩缝合术等

👁️ 5.3 结膜炎（Conjunctivitis）

📌 一、定义

结膜因刺激或感染发生的炎症，伴充血、分泌物增加。

⚙️ 二、病因

  • 机械性刺激（灰尘、异物）
  • 感染性（细菌、病毒，如犬瘟、猫疱疹）
  • 过敏性、免疫反应、继发于其他眼病

👀 三、临床表现

  • 流泪、羞明、结膜红肿、黏液或脓性分泌物
  • 急性期：明显充血；慢性期：结膜增厚、滤泡形成

🛠️ 四、治疗

  • 去除诱因，保持局部清洁
  • 使用抗菌滴眼液或眼膏
  • 严重者配合抗生素或激素类药物局部应用

👁️ 5.4 角膜炎（Keratitis）

📌 一、定义

角膜组织发生的非化脓或化脓性炎症，影响透明性和视力。

⚙️ 二、病因

  • 创伤、异物、睫毛倒生
  • 传染病（犬传染性肝炎）、代谢障碍
  • 干眼症导致继发性角膜炎

👀 三、临床表现

  • 流泪、羞明、角膜混浊、血管新生
  • 严重者出现角膜溃疡、穿孔

🛠️ 四、治疗

  • 抗生素/抗病毒滴眼液，促进角膜修复
  • 干眼相关者联合泪液替代剂、免疫抑制剂
  • 溃疡严重者需手术如第三眼睑覆盖术或角膜移植

👁️ 5.5 瞬膜腺突出（Cherry Eye）

📌 一、定义

瞬膜（第三眼睑）中的腺体增大并脱出至眼睑边缘，外观似“樱桃”。

图片：瞬膜腺突出病例


瞬膜是位于犬、猫眼内眦薄膜状组织瞬膜保护角膜，同时其内的腺体还有分泌泪液的功能。瞬膜腺分泌提供了占眼球表面泪液层的泪液，如果不做治疗或者做传统的切除术，患病动物后续发生干性角膜结膜炎的概率会增高。

⚙️ 二、病因

  • 先天结构松弛
  • 遗传倾向（如可卡犬、斗牛犬）
  • 瞬膜腺组织结构发育异常

👀 三、临床表现

  • 红色肿块突现于内眼角
  • 有结膜刺激、流泪、继发性结膜炎风险

🛠️ 四、治疗

  • 不建议直接切除，易导致干眼
  • 采用“口袋缝合法”或“锚定术”复位腺体并保留功能

👁️ 5.6 虹膜炎（Iritis）

📌 一、定义

虹膜（眼色素层）炎症，通常伴有前房变化，影响眼压和视力。

⚙️ 二、病因

  • 创伤、穿透性角膜炎
  • 系统性疾病（如犬瘟热）或邻近组织感染蔓延

👀 三、临床表现

  • 前房混浊、虹膜颜色改变
  • 瞳孔缩小、粘连，羞明、流泪
  • 严重时出现继发性青光眼或白内障

🛠️ 四、治疗

  • 阿托品散瞳，防止虹膜粘连
  • 应用局部糖皮质激素及抗生素
  • 控制原发病因和防止并发症

👁️ 5.7 视网膜炎（Retinitis）

📌 一、定义

视网膜发生炎症，常为感染或毒素引起，影响视力。

⚙️ 二、病因

  • 病毒性感染（如犬瘟热）
  • 自身免疫疾病、寄生虫感染、全身感染转移

👀 三、临床表现

  • 视力下降或失明
  • 眼底检查见视网膜水肿、出血、渗出
  • 视神经乳头边缘模糊

🛠️ 四、治疗

  • 原因治疗（控制感染）
  • 局部或全身使用抗生素、糖皮质激素
  • 营养支持、保护视神经功能

👁️ 5.8 青光眼（Glaucoma）

📌 一、定义

青光眼是由于眼压持续升高，导致视神经受损、视网膜变性，最终引起视力丧失的一种眼病。其特点是眼球胀大、角膜混浊和疼痛明显。

⚙️ 二、病因

  • 原发性：眼房角异常、先天性房水排出障碍
  • 继发性：虹膜炎、晶状体脱位、眼内肿瘤压迫房角
  • 先天性畸形：房水引流系统发育不良
  • 创伤性：穿透伤、出血引起房角闭塞

视频：青光眼原来是这样形成的

👀 三、临床表现

  • 眼压升高（犬＞25 mmHg，猫＞21 mmHg）
  • 角膜水肿、混浊、扩瞳、眼球突出
  • 瞳孔对光反射迟钝或消失，视力下降
  • 慢性期视神经萎缩，眼底呈“凹陷”
  • 行为异常如怕光、焦躁、持续抓眼等

🧪 四、诊断

  • 眼压测定：回弹式眼压计、Tono-Pen等
  • 眼底镜检查：视神经盘凹陷
  • 裂隙灯：前房深浅、角膜透明度变化
  • 荧光素试验：排除角膜病变

💊 五、治疗

  • 降低眼压

    • 利尿剂：20%甘露醇（1-2 g/kg静注）
    • 碳酸酐酶抑制剂：乙酰唑胺（口服）
    • 前列腺素类似物：拉坦前列素滴眼
    • β-阻滞剂：噻吗心安、贝他洛尔滴眼

  • 促进房水排出

    • 缩瞳剂（毛果芸香碱）增强房角引流

  • 手术

    • 眼前房分流术、睫状体冷冻治疗、激光房角切除术

  • 晚期/不可逆失明

    • 眼球摘除术或眼球内义眼植入术

👁️ 5.9 白内障（Cataract）

📌 一、定义

白内障是由于晶状体蛋白变性或脱水导致的晶状体混浊，影响光线通过并最终造成视力下降甚至失明。

⚙️ 二、病因

  • 先天性：出生后不久出现，可能伴眼球发育异常
  • 遗传性：如西施、比熊、贵宾犬等品种高发
  • 老年性：晶状体代谢退化（8岁以上多见）
  • 继发性：糖尿病性、外伤性、慢性葡萄膜炎、青光眼
  • 营养性：幼龄犬猫断奶期营养不良（缺乏胱氨酸、锌等）

📅 三、分类（按发病过程）

  • 初发期（初期混浊）
  • 不成熟期（部分区域变浑）
  • 成熟期（完全混浊）
  • 过熟期（晶状体液化）

👀 四、临床表现

  • 瞳孔区域出现灰白、乳白色混浊斑块
  • 对光反应减弱，视物能力下降甚至失明
  • 不伴发炎症，但常与葡萄膜炎、青光眼并存

🧪 五、诊断

  • 裂隙灯显微镜：可见晶状体皮质/核混浊
  • 检影镜观察：晶状体混浊呈灰白色不透明
  • 超声波（B超）：用于成熟性白内障看清玻璃体及视网膜情况

💊 六、治疗

非手术（早期）

  • 白内障尚未成熟时，使用药物改善代谢
    • 吡诺克辛滴眼液
    • 法可林（促进蛋白代谢）
    • 维生素C、E、B族口服辅助

手术治疗（首选）

  • 囊外晶状体摘除术（ECCE）
  • 超声乳化术（phacoemulsification）：晶状体吸出后植入人工晶体
  • 手术适应症：晶状体混浊明显、眼内无炎、视网膜功能存在

☑️ 七、术后管理

  • 严格抗炎：地塞米松、妥布霉素滴眼
  • 控制眼压，避免术后并发青光眼
  • 防止感染、粘连和角膜水肿
