第五章 🐾 眼科

👁️ 5.1 眼的解剖生理

视频：【绘画解剖学 眼球】眼球的解剖与视力调节

✨ 一、眼球组成与功能

眼由眼球及其附属结构构成，其功能由以下结构完成：

  • 感光结构：视网膜的视锥细胞与视杆细胞 → 视神经 → 视觉中枢
  • 屈光结构：角膜、眼房液、晶状体、玻璃体
  • 营养结构：葡萄膜、血管、房水
  • 保护结构：眼睑、结膜、泪器、角膜、巩膜、眼眶
  • 运动结构：眼球直肌、斜肌、退缩肌

🧱 二、眼球壁结构（外、中、内三层）

🟩 外层：纤维膜

  • 角膜：透明、无血管、有强屈光力
  • 巩膜：乳白色坚韧结缔组织，血供丰富

🟧 中层：葡萄膜

  • 虹膜：含瞳孔括约肌与开大肌，调节光线进入
  • 睫状体：分泌房水，调节晶状体形状
  • 脉络膜：营养视网膜外层，含照膜（反光结构）

🟥 内层：视网膜

  • 感光部分包括视锥和视杆细胞
  • 中央有“视神经乳头” → 无感光能力，称为盲点

💧 三、眼球内容物

  • 房水：分泌→循环→吸收，维持眼压
  • 晶状体：双凸透明体，改变形状调节焦距
  • 玻璃体：透明凝胶体，维持眼球形态与屈光性

👁 四、眼附属器

  • 眼睑：保护眼球、分泌脂质保持湿润
  • 结膜：透明黏膜，覆盖眼球前部与眼睑后面
  • 泪器：泪腺+泪道（泪点→泪小管→泪囊→鼻泪管）
  • 眼外肌：6条眼球肌 + 1条退缩肌 + 上睑提肌等
  • 眼眶：骨性结构，保护眼球

⚡ 五、眼的神经与血液供应

  • 动脉：主要来自颈内动脉系统
  • 静脉：中央视网膜静脉、涡静脉、睫状前静脉
  • 神经：动眼、滑车、外展、三叉神经各支配运动与感觉功能

🌈 六、感光作用机制

  • 屈光装置：调节晶状体折光 → 成像于视网膜
  • 瞳孔反射：控制入光量
  • 感光细胞分布：犬猫色盲但暗视觉发达

🔍 5.2 眼的检查方法

🩺 一、一般检查法

👁 1. 视诊

  • 检查眼睑、结膜、角膜、巩膜、前房、虹膜、瞳孔、晶状体等外观变化

✋ 2. 触诊

  • 检查眼睑肿胀、温度、压痛、眼压变化

🧪 二、器械检查方法

  1. 角膜镜检查：观察角膜曲率、透明度（同心圆反光判断）
  2. 烛光映像检查：正常见3个光影
  3. 检眼镜检查：查看玻璃体、视网膜血管、绿毯/暗毯等
  4. 裂隙灯显微镜：检查前节微小病变
  5. 眼压测定：回弹式眼压计（猫犬正常值见文中）
  6. 荧光素染色法：检查角膜溃疡、鼻泪管通畅
  7. Schirmer氏泪液试验：检测泪液分泌量
  8. 影像学检查：鼻泪管造影、眼底照相、B超、CT、MRI等

💊 5.3 眼科用药与治疗技术

💧 一、常用药物分类

  1. 洗眼液：2%硼酸、0.9%氯化钠
  2. 收敛/腐蚀药：硫酸锌、硝酸银、蛋白银等
  3. 抗菌药：磺胺类、庆大霉素、金霉素等
  4. 抗病毒/抗真菌药：阿昔洛韦、氟康唑
  5. 抗炎药
    • 类固醇类（泼尼松龙、地塞米松）
    • 非甾体类（卡洛芬、酮洛芬）
  6. 散瞳药/缩瞳药：阿托品、毛果芸香碱
  7. 免疫调节剂：环孢素、咪唑类、贝复舒等
  8. 青光眼用药：甘露醇、拉坦前列素、β阻滞剂
  9. 白内障用药：吡诺克辛、法可林

🛠️ 二、常用治疗技术

  • 洗眼、点眼、眼膏涂布
  • 结膜下注射、眼内注射
  • 球后麻醉、眼睑下灌流法
  • 治疗性软性隐形眼镜（角膜绷带）
  • 放疗、热疗、激光术
